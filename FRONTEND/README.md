# To-Do List
This is a full-stack, simulated to-do list created to learn and demonstrate various web technologies. It includes a backend and a separate frontend application.

## Built with
- React
- Bootstrap
- Vite
- Django
- CORDS

## Getting Started
### Installation
1. Clone the repository [https://github.com/LisAY22/To-DoList.git](https://github.com/LisAY22/To-DoList.git)
2. Follow the instructions in the README.md files

To run this project, you must run both the backend API and the frontend application in two separate terminals.
