# To-Do List Frontend
This project is a frontend user interface for the "To-Do List" project.
It connects to its own API to fetch tasks data, create them or update them. 

## Features
- View all tasks.
- Filter tasks in real-time.
- Add new tasks.
- Update state of tasks. 

## Technologies
This project was built with:
- Node.js: JavaScript runtime environment
- React (Vite): A JavaScript library for building user interfaces.
- Bootstrap: CSS framework for styling and layout.

## How to Get Started
### Prerequisites
- Node.js 
- The To-Do List Backend must be running on for this app to fetch data.

### Installation & Setup
1. Clone the repository 
[https://github.com/LisAY22/To-DoList.git](https://github.com/LisAY22/To-DoList.git)
2. Navigate to the API project directory 
```cd TO-DOLIST/FRONTEND/todoFront```
3. Install all the required dependencies
```npm install```

### Run the server
```npm run dev```

Vite will start the development server. Check your terminal for the local URL and open it in your browser.
